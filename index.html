Simple Viewer 3D

A simple, fast, open source, 3D file viewer that can open:

  • .stl files
  • .obj files

Currently, you can only download it for Windows x86.

Github git

You can compile to ARM on windows using the Github and following the README. Just keep in mind the ARM build has not been tested yet (which is why a download for it is not published).

Features:

  • Multiple Files and tabs open at once.
  • Window docking
  • Fast file load times
  • Fast renderer using
